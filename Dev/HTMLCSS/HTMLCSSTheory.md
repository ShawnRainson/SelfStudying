##📚 Теория##
•	HTML (HyperText Markup Language) — это не язык программирования. Здесь нет логики, условий или вычислений. Это язык разметки, который создает структуру и каркас веб-страницы с помощью тегов.
•	Как устроен Web: Браузер запрашивает файл index.html с сервера, читает его сверху вниз и строит в своей оперативной памяти дерево объектов — DOM (Document Object Model). Каждый тег (заголовок, абзац, картинка) становится отдельным узлом этого дерева. Браузер всегда пытается отобразить страницу, даже если ты забыл закрыть тег или допустил ошибку — он не упадет с ошибкой, а попытается «догадаться», что ты имел в виду, из-за чего верстка часто «плывет».

#🌐 Трек 2: HTML/CSS (Подключение стилей)#
##📚 Теория##
HTML отвечает только за скелет. Вся визуальная магия (цвета, отступы, размеры) выносится в отдельный файл CSS, чтобы разделять структуру и дизайн.
🔬 Пример синтаксиса (Селекторы и свойства)
Вот пример CSS-кода, который делает задний фон страницы серым, красит ссылки в зелёный и увеличивает текст списков:
css
```
body {
    background-color: gray;
}

a {
    color: green;
}

li {
    font-size: 20px;

``

# 🎨 HTML/CSS: Селекторы классов (День 17)

### 📚 Теория
Чтобы стилизовать конкретные элементы, а не все теги одного типа сразу, им задают индивидуальные или групповые классы через атрибут `class=""` в HTML. В CSS селектор для стилизации класса всегда начинается с точки `.`. Это позволяет применять разные стили к одинаковым тегам (например, делать абзацы разного цвета).

### 🔬 Пример синтаксиса
**HTML:**
```html
<p class="warning-text">Внимание!</p>
```
**CSS:**
```css
.warning-text {
    color: orange;
    font-weight: bold;
}
```

### 🛠 Задание
1. Добавь в свой рабочий HTML-файл два новых абзаца `<p>`. Первому задай атрибут `class="success"`, второму — `class="danger"`.
2. В файле `style.css` напиши стили так, чтобы класс `success` красил текст в зеленый цвет (`color: green;`), а класс `danger` — в красный (`color: red;`). Открой страницу в браузере и проверь результат.
