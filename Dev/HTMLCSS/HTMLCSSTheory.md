##📚 Теория##
•	HTML (HyperText Markup Language) — это не язык программирования. Здесь нет логики, условий или вычислений. Это язык разметки, который создает структуру и каркас веб-страницы с помощью тегов.
•	Как устроен Web: Браузер запрашивает файл index.html с сервера, читает его сверху вниз и строит в своей оперативной памяти дерево объектов — DOM (Document Object Model). Каждый тег (заголовок, абзац, картинка) становится отдельным узлом этого дерева. Браузер всегда пытается отобразить страницу, даже если ты забыл закрыть тег или допустил ошибку — он не упадет с ошибкой, а попытается «догадаться», что ты имел в виду, из-за чего верстка часто «плывет».

#🌐 Трек 2: HTML/CSS (Подключение стилей)#
##📚 Теория##
HTML отвечает только за скелет. Вся визуальная магия (цвета, отступы, размеры) выносится в отдельный файл CSS, чтобы разделять структуру и дизайн.
🔬 Пример синтаксиса (Селекторы и свойства)
Вот пример CSS-кода, который делает задний фон страницы серым, красит ссылки в зелёный и увеличивает текст списков:
css
```
body {
    background-color: gray;
}

a {
    color: green;
}

li {
    font-size: 20px;

``

# 🎨 HTML/CSS: Селекторы классов (День 17)

### 📚 Теория
Чтобы стилизовать конкретные элементы, а не все теги одного типа сразу, им задают индивидуальные или групповые классы через атрибут `class=""` в HTML. В CSS селектор для стилизации класса всегда начинается с точки `.`. Это позволяет применять разные стили к одинаковым тегам (например, делать абзацы разного цвета).

### 🔬 Пример синтаксиса
**HTML:**
```html
<p class="warning-text">Внимание!</p>
```
**CSS:**
```css
.warning-text {
    color: orange;
    font-weight: bold;
}
```

### 🛠 Задание
1. Добавь в свой рабочий HTML-файл два новых абзаца `<p>`. Первому задай атрибут `class="success"`, второму — `class="danger"`.
2. В файле `style.css` напиши стили так, чтобы класс `success` красил текст в зеленый цвет (`color: green;`), а класс `danger` — в красный (`color: red;`). Открой страницу в браузере и проверь результат.

# 🎨 HTML/CSS: Идентификаторы (ID) элементов (День 18)

### 📚 Теория
Мы уже знаем, что классы (`class`) нужны для стилизации *группы* элементов. Но если элемент на странице **абсолютно уникален** (например, главная шапка сайта, логотип или кнопка отправки формы), ему задают идентификатор через атрибут `id=""`. 
* Важное правило: один и тот же `id` может встретиться на HTML-странице **только один раз**.
* В CSS селектор идентификатора всегда начинается со знака решётки `#`.

### 🔬 Пример синтаксиса
**HTML:**
```html
<div id="main-header">Welcome to my site</div>
```
**CSS:**
```css
#main-header {
    background-color: darkblue;
    color: white;
}
```

### 🛠 Задание
1. Добавь в свой HTML-файл один уникальный тег `<div>` (блок) или `<h1>` и задай ему `id="author-signature"`. Напиши внутри своё имя.
2. В файле `style.css` через селектор `#` задай для этого элемента уникальный цвет текста и сделай его курсивным (`font-style: italic;`).

# 🎨 HTML/CSS: Псевдоклассы (:hover) (День 19)

### 📚 Теория
Псевдоклассы в CSS позволяют менять внешний вид элементов, когда пользователь совершает с ними какое-то действие. Самый популярный псевдокласс — это `:hover`. Стили внутри него срабатывают в тот момент, когда на элемент **наводят курсор мыши**.

### 🔬 Пример синтаксиса
```css
button {
    background-color: gray;
}
button:hover {
    background-color: blue; /* Цвет изменится при наведении */
}
```

### 🛠 Задание
1. В своем файле `style.css` найди селектор для идентификатора `#author-signature`.
2. Допиши ниже новый селектор с псевдоклассом `#author-signature:hover`.
3. Сделай так, чтобы при наведении мышки цвет твоего имени менялся с оранжевого на любой другой (например, красный или фиолетовый), а размер шрифта увеличивался. Открой страницу в браузере и поводи мышкой по имени!
