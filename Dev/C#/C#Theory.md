#🏎 Трек 1: C# (Си-Шарп)#
##📚 Теория##
C# — это современный объектно-ориентированный язык от Microsoft.
•	Строгая типизация: Каждая переменная имеет четкий тип (число, текст, булево значение). Смешивать их нельзя. Если ты объявил переменную для возраста, засунуть туда строку "тридцать" не получится — компилятор заблокирует сборку. Это предотвращает баги на этапе написания кода.
•	Как это работает под капотом: Код не выполняется напрямую процессором. Сначала компилятор превращает твой код в промежуточный язык (IL — Intermediate Language). Когда пользователь запускает программу, виртуальная машина .NET CLR (Common Language Runtime) «на лету» переводит IL-код в машинные инструкции конкретного процессора. Также CLR сама управляет памятью через Garbage Collector (сборщик мусора), который автоматически удаляет ненужные объекты из ОЗУ.

# 🏎 C#: Условный оператор if-else (День 17)

### 📚 Теория
Конструкция `if`, `else if` и `else` управляет логикой программы на основе условий. Условия обязательно пишутся в круглых скобках `()`, а блоки кода — в фигурных `{}`. Для объединения нескольких условий в один логический блок используется оператор «И» (`&&`).

### 🔬 Пример синтаксиса
```csharp
int fuel = 15;

if (fuel == 0) {
    Console.WriteLine("Engine stopped!");
}
else if (fuel < 20) {
    Console.WriteLine("Low fuel warning!");
}
else {
    Console.WriteLine("Fuel level is normal.");
}
```

### 🛠 Задание
1. Внутри метода `Main` создай переменную `score` (твои очки в тесте) типа `int` и присвой ей любое значение от 0 до 100.
2. Напиши условие: если `score` больше или равен 90, выведи в консоль `"Excellent!"`. Если он находится в диапазоне от 50 до 89 включительно (используй `&&`), выведи `"Good job!"`. В любых других случаях выведи `"Try again!"`.

# 🏎 C#: Массивы и цикл foreach (День 18)

### 📚 Теория
Когда нужно хранить много однотипных данных (например, список оценок или имен), используют **массивы**. Квадратные скобки `[]` после типа данных показывают, что это массив. Чтобы быстро перебрать все элементы массива по очереди, используется очень удобный цикл `foreach`.

### 🔬 Пример синтаксиса
```csharp
string[] cars = { "BMW", "Audi", "Tesla" };

foreach (string car in cars) {
    Console.WriteLine(\$"Car model: {car}");
}
```

# 🏎 C#: Знакомство с интерфейсами (interface) (День 19)

### 📚 Теория
Интерфейс в ООП — это строгий контракт. Он не содержит готового кода, а только объявляет, какие методы *обязан* иметь класс, который этот интерфейс реализует. Интерфейсы всегда начинаются с заглавной буквы `I`. Они нужны для того, чтобы связать разные классы единым стандартом.

### 🔬 Пример синтаксиса
```csharp
interface IVehicle {
    void Move(); // Только объявление метода, без {}
}

class Car : IVehicle { // Класс обязан реализовать этот метод
    public void Move() {
        Console.WriteLine("Car is driving");
    }
}
```

### 🛠 Задание
1. Создай интерфейс `IReviewable`. Внутри него объяви один метод `void Review();` (без реализации).
2. Сделай так, чтобы твой класс `Program` реализовал этот интерфейс. Добавь в класс public-метод `Review()`, который выводит в консоль фразу: `"Code review is passed!"`.
3. В методе `Main` создай объект класса `Program` и вызови у него метод `Review()`.

# 🏎 C#: Динамические списки List<T> (День 20)

### 📚 Теория
Обычные массивы (`int[]`) имеют фиксированную длину — её нельзя изменить после создания. Если нужно динамически добавлять или удалять элементы в процессе работы программы, используют списки `List<T>`. Чтобы использовать их, в самом верху файла обязательно должна быть подключена библиотека `using System.Collections.Generic;`.

### 🔬 Пример синтаксиса
```csharp
List<string> names = new List<string>();
names.Add("Ivan"); // Добавление элемента
names.Remove("Ivan"); // Удаление элемента
Console.WriteLine(names.Count); // Получить размер списка
```

1. Подключи пространство имен `using System.Collections.Generic;`.

4. Используя знакомый цикл `foreach`, выведи весь список покупок в консоль.

# 🏎 C#: Продвинутые методы строк и StringBuilder (День 21)

### 📚 Теория
В C# строки (`string`) являются неизменяемыми (**immutable**). Каждый раз, когда ты делаешь `str += " ещё текст"`, в памяти создается новый объект, а старый выбрасывается. Если собирать длинный текст или массив слов в цикле, это сильно бьет по производительности. Для эффективной сборки длинных текстов используют специальный класс `StringBuilder` из пространства `System.Text`.

### 🔬 Пример синтаксиса
```csharp
using System.Text;

StringBuilder sb = new StringBuilder();
sb.Append("Hello ");
sb.Append("World!");
string result = sb.ToString(); // Конвертируем в обычную строку
```

### 🛠 Задание
1. Подключи пространство имен `using System.Text;`.
2. Внутри метода `Main` создай массив строк `words`, содержащий 4 любых ИТ-термина (например, `"Loop"`, `"Class"`, `"Method"`, `"Array"`).
3. Создай объект `StringBuilder`. С помощью цикла `foreach` обойди массив и добавь каждое слово в `StringBuilder`, разделяя их запятой и пробелом.
4. Выведи финальную собранную строку в консоль.


# 🏎 C#: Работа со временем через DateTime (День 22)

### 📚 Теория
В C# для работы с датами и временем используется структура `DateTime`. С её помощью можно получить текущее время компьютера (`DateTime.Now`), прибавлять дни, часы или форматировать вывод даты в текст.

### 🔬 Пример синтаксиса
```csharp
DateTime current = DateTime.Now;
Console.WriteLine(current.ToString("dd.MM.yyyy")); // Выведет дату в формате ДД.ММ.ГГГГ
```
