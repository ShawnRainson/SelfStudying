#🏎 Трек 1: C# (Си-Шарп)#
##📚 Теория##
C# — это современный объектно-ориентированный язык от Microsoft.
•	Строгая типизация: Каждая переменная имеет четкий тип (число, текст, булево значение). Смешивать их нельзя. Если ты объявил переменную для возраста, засунуть туда строку "тридцать" не получится — компилятор заблокирует сборку. Это предотвращает баги на этапе написания кода.
•	Как это работает под капотом: Код не выполняется напрямую процессором. Сначала компилятор превращает твой код в промежуточный язык (IL — Intermediate Language). Когда пользователь запускает программу, виртуальная машина .NET CLR (Common Language Runtime) «на лету» переводит IL-код в машинные инструкции конкретного процессора. Также CLR сама управляет памятью через Garbage Collector (сборщик мусора), который автоматически удаляет ненужные объекты из ОЗУ.

# 🏎 C#: Условный оператор if-else (День 17)

### 📚 Теория
Конструкция `if`, `else if` и `else` управляет логикой программы на основе условий. Условия обязательно пишутся в круглых скобках `()`, а блоки кода — в фигурных `{}`. Для объединения нескольких условий в один логический блок используется оператор «И» (`&&`).

### 🔬 Пример синтаксиса
```csharp
int fuel = 15;

if (fuel == 0) {
    Console.WriteLine("Engine stopped!");
}
else if (fuel < 20) {
    Console.WriteLine("Low fuel warning!");
}
else {
    Console.WriteLine("Fuel level is normal.");
}
```

### 🛠 Задание
1. Внутри метода `Main` создай переменную `score` (твои очки в тесте) типа `int` и присвой ей любое значение от 0 до 100.
2. Напиши условие: если `score` больше или равен 90, выведи в консоль `"Excellent!"`. Если он находится в диапазоне от 50 до 89 включительно (используй `&&`), выведи `"Good job!"`. В любых других случаях выведи `"Try again!"`.

# 🏎 C#: Массивы и цикл foreach (День 18)

### 📚 Теория
Когда нужно хранить много однотипных данных (например, список оценок или имен), используют **массивы**. Квадратные скобки `[]` после типа данных показывают, что это массив. Чтобы быстро перебрать все элементы массива по очереди, используется очень удобный цикл `foreach`.

### 🔬 Пример синтаксиса
```csharp
string[] cars = { "BMW", "Audi", "Tesla" };

foreach (string car in cars) {
    Console.WriteLine(\$"Car model: {car}");
}
```

### 🛠 Задание
1. Внутри метода `Main` создай массив целых чисел `grades`, в который запиши 5 любых оценок (например, от 2 до 5).
2. Используя цикл `foreach`, пройдись по всему массиву и выведи каждую оценку в консоль.
3. *Усложнение (по желанию):* Попробуй внутри цикла посчитать сумму всех этих оценок и вывести её на экран после завершения цикла.
