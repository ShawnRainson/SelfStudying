# ☕ JavaScript: Ветвление if-else (День 17)

### 📚 Теория
Синтаксис ветвления в JS практически идентичен C#. Условия пишутся в круглых скобках `()`, блоки кода — в фигурных `{}`. Для строгого сравнения (когда проверяется одновременно и значение, и тип данных) в JS всегда используется тройное равенство `===`. Использование двойного `==` в современном коде не рекомендуется, так как оно может приводить к скрытым ошибкам из-за автоматического приведения типов.

### 🔬 Пример синтаксиса
```javascript
let userStatus = "admin";

if (userStatus === "admin") {
    console.log("Full access granted");
} else {
    console.log("Limited access");
}
```

### 🛠 Задание
1. Создай переменную `temperature` через `let` и присвой ей значение `15`.
2. Напиши условие `if-else`: если температура меньше 10, выведи в консоль браузера строку `"It's cold"`. Иначе (в любом другом случае) выведи `"It's warm"`.

# ☕ JavaScript: Массивы и цикл for...of (День 18)

### 📚 Теория
В JavaScript массивы создаются с помощью квадратных скобок `[]` и могут динамически изменять свой размер. Для простого перебора элементов массива (аналог `foreach` в C#) в современном JS используется цикл `for...of`.

### 🔬 Пример синтаксиса
```javascript
let fruits = ["Apple", "Banana", "Orange"];

for (let fruit of fruits) {
    console.log("Fruit: " + fruit);
}
```

# ☕ JavaScript: Работа со временем (setTimeout) (День 19)

### 📚 Теория
Иногда код нужно выполнить не моментально, а с задержкой (например, показать всплывающее окно через 3 секунды). Для этого в JS есть встроенная функция `setTimeout()`. Она принимает два параметра: функцию, которую нужно выполнить, и время задержки в **миллисекундах** (1 секунда = 1000 мс).

### 🔬 Пример синтаксиса
```javascript
setTimeout(function() {
    console.log("Прошло 2 секунды");
}, 2000);
```

### 🛠 Задание
1. Внутри тега `<script>` напиши функцию `setTimeout`.
2. Настрой задержку ровно на **4000 миллисекунд** (4 секунды).
3. Сделай так, чтобы по истечении этого времени в консоль браузера выводилась строка: `"4 seconds left. Task completed!"`.

# ☕ JavaScript: Чтение данных из Input и динамический вывод (День 21)

### 📚 Теория
Теперь мы соединим поле ввода и кнопку. С помощью свойства `.value` JavaScript может мгновенно прочитать текст, который пользователь написал в поле `<input>`. А с помощью событий мы можем отследить этот ввод при нажатии на кнопку.

### 🔬 Пример синтаксиса
```javascript
let inputField = document.getElementById("input-id");
let text = inputField.value; // Получаем текущий текст из инпута
```

# ☕ JavaScript: Защита кода через try...catch (День 22)

### 📚 Теория
В JS, как и в Python, код может упасть из-за непредвиденной ошибки (например, попытка прочитать свойство у несуществующего элемента). Чтобы скрипт не прекращал работу, опасные участки оборачивают в блок `try...catch`.

### 🔬 Пример синтаксиса
```javascript
try {
    undefinedFunction(); // Ошибка!
} catch (error) {
    console.log("Поймали ошибку в JS: " + error.message);
}
```
