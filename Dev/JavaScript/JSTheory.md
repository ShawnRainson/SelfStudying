# ☕ JavaScript: Ветвление if-else (День 17)

### 📚 Теория
Синтаксис ветвления в JS практически идентичен C#. Условия пишутся в круглых скобках `()`, блоки кода — в фигурных `{}`. Для строгого сравнения (когда проверяется одновременно и значение, и тип данных) в JS всегда используется тройное равенство `===`. Использование двойного `==` в современном коде не рекомендуется, так как оно может приводить к скрытым ошибкам из-за автоматического приведения типов.

### 🔬 Пример синтаксиса
```javascript
let userStatus = "admin";

if (userStatus === "admin") {
    console.log("Full access granted");
} else {
    console.log("Limited access");
}
```

### 🛠 Задание
1. Создай переменную `temperature` через `let` и присвой ей значение `15`.
2. Напиши условие `if-else`: если температура меньше 10, выведи в консоль браузера строку `"It's cold"`. Иначе (в любом другом случае) выведи `"It's warm"`.
