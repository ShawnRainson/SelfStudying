Задание №1

Напиши программу "Каталог фильмов".

Каждый фильм должен хранить:

название;
год;
жанр;
рейтинг.

Используй:

список;
словари;
цикл;
функции.

Меню:

1. Добавить фильм

2. Показать все фильмы

3. Найти фильм

4. Удалить фильм

5. Выход
Требования

Не использовать классы.

Почему?

Потому что сначала нужно научиться уверенно работать с коллекциями.
